Link de los comandos => https://docs.djangoproject.com/en/3.2/ref/django-admin/
django-admin startproject <project_name> => crea un proyecto con ese nombre
python manage.py runserver => levanta el servidor
python manage.py showmigrations => lista todas las migraciones que hay en el proyecto y las que han sido efectuadas estaran marcadas con una X
python manage.py migrate => efectua todas las migraciones que aun no se han realizado en la bd
python manage.py startapp libreria => crea una aplicacion para que podamos crear modelos, vistas, rutas, entre otros
python manage.py makemigrations <nombre_app> => registra los cambios realizados en los models para que puedan ser ejecutados en la bd
python manage.py createsuperuser => crea un super usuario que se registrara en la tabla auth_user y servira para poder acceder al panel administrativo
python manage.py sqlmigrate <nombre_app> <num_migracion> => nos da el codigo SQL que se ejecutara en la base de datos a partir de la migracion
python manage.py shell => crea una terminal para trabajar con python Pero usando todas las funcionalidades de Django
python manage.py collectstatic => genera todos los archivos CSS, JS y HTML que se utilizan de manera interna en el proyecto (DRF, DJANGO)
